=head2 Add a new entry type

to add a new type "RTx" with an extra meta key "category", you just need to
create a module Beagle::Model::RTx which inherits from Beagle::Model::Entry,
then update beagle's cache, that's it.

    package Beagle::Model::RTx;
    use Any::Moose;
    extends 'Beagle::Model::Entry';
    
    has 'category' => (
        isa     => 'Str',
        is      => 'rw',
        default => '',
    );
    
    override 'serialize_meta' => sub {
        my $self = shift;
        my $str  = super;
        $str .= $self->_serialize_meta( 'category' );
        return $str;
    };

    1;


    $ beagle cache -ufa

now you have rtx and rtxes commands automatically.
